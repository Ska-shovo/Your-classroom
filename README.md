## Overview
An AI-powered classroom monitoring platform designed to enhance student engagement, track attendance, analyze classroom participation, and provide real-time insights for teachers during online learning sessions.

## Key Features
- Real-time Attendance Monitoring
- Student Engagement Analysis
- Camera Status Detection
- Classroom Performance Analytics
- Teacher Dashboard
- AI-Based Participation Tracking
- Automated Reports and Insights

## Technologies Used
- HTML, CSS, JavaScript
- React.js
- Node.js
- Express.js
- MongoDB
- WebRTC
- AI/ML Models
- TensorFlow / OpenCV

## Project Objectives
- Enhance student engagement in online learning.
- Increase active participation during virtual classes.
- Provide real-time classroom monitoring.
- Enable teachers to monitor attendance and participation.
- Generate performance analytics and reports.
- Improve overall online learning effectiveness.
